Installing

Method 1

Download and install deb packages from https://github.com/hawkeye116477/waterfox-deb/tree/master/pool/main.

Method 2

Run these commands to add this repostory to your APT sources list: 

$ echo "deb https://hawkeye116477.github.io/waterfox-deb release main" | \
sudo tee /etc/apt/sources.list.d/waterfox-release-hawkeye116477.list

And the public GPG key: 

$ curl https://hawkeye116477.github.io/waterfox-deb/hawkeye116477.key | 
sudo apt-key add -

You can download and install the packages after you update the APT package index: 

$ sudo apt-get update

Note: Language packs and KDE integration features are available as separate packages to install from apt repository!

Source

Waterfox packages were generated with files from official Waterfox website.

Waterfox KDE Plasma Edition was generated with files from official Waterfox GitHub and contains openSUSE's KDE patches and fix-wifi-scanner.diff.

Packages: gcc-6-base, libgcc1 and libstdc++6 are from official Ubuntu Yakkety repository.
